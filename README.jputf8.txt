CryptMT version 3
eSTREAM に参加したストリーム暗号 CryptMT3 のC++ ソースファイルです。
Windows 版は dll ファイルもあります。

ライセンスについては LICENSE.txt をご覧下さい。

C++ の CryptMT クラスのドキュメントは doc/jp/index.html に
あります。

CryptMT クラスの使用例として cryptfile という実行形式も作れるように
なっています。
cryptfile は入力ファイルを暗号化して出力するプログラムです。

./cryptfile [-e|-d] [-i ifile] [-o ofile] [-k key] [-f kfile][-s size] [-v]

-e           暗号化
-d           復号
             省略時は暗号化する。とはいえ、暗号化と復号は同じである。
             （ストリーム暗号なので）
-i ifile     入力ファイル. 省略時は標準入力が使用される
-o ofile     出力ファイル. 省略字は標準出力が使用される
-k key       暗号化および復号キー
-f kfile     暗号化および復号キーを含むファイル
             -k と -f のどちらも指定されないときはキーの入力
             が求められる
             両方指定されたときは、 -k の値が使用される
-s size      キーのサイズ（ビット） キーサイズは128以上で2048以下でなければ
             ならない。省略字は128が使用される。
-v           詳細メッセージ（暗号化時間）が表示される
